# Getting Support

How to get help with Shadow Clone.

---

## Before Contacting Support

### 1. Check the Documentation

- [Common Issues](common-issues.md) - FAQ and solutions
- [Error Codes](../reference/error-codes.md) - Specific error solutions
- [Installation Guide](../getting-started/installation.md) - Setup help

### 2. Gather Information

Before reaching out, collect:

- **Error message** (exact text)
- **Node.js version**: `node --version`
- **OS and version**: macOS 14.0, Windows 11, Ubuntu 22.04, etc.
- **Claude version**: Claude Code or Claude Desktop
- **License tier**: Elite, Pioneer, or Builder
- **Steps to reproduce**: What were you doing?

### 3. Try Basic Fixes

1. Restart Claude
2. Re-authenticate
3. Check configuration file
4. Reinstall MCP server

## Contact Options

### Discord Community

**Best for**: Quick questions, community help, general discussion

- Join the Ignis Labs Discord
- Post in #shadow-clone-support channel
- Community members and team can help

### Email Support

**Best for**: Account issues, licensing questions, private concerns

- Email: support@ignislabs.ai
- Include your wallet address (for license verification)
- Response time: 24-48 hours

### Dashboard

**Best for**: API key management, license status

- Visit [dashboard.ignislabs.ai](https://dashboard.ignislabs.ai)
- Connect your wallet
- View license status
- Generate new API keys

## Reporting Bugs

### Where to Report

GitHub Issues (for technical bugs):
- Open an issue with detailed reproduction steps
- Include error messages and logs
- Mention your environment (OS, Node version, etc.)

## License Support

### Common License Questions

**Q: How do I check my license tier?**

Use `api_key_status()` or visit dashboard.ignislabs.ai

**Q: How do I upgrade my license?**

Purchase a higher-tier NFT from the Ignis Labs collection.

**Q: My NFT transferred, what now?**

Connect the new wallet to the dashboard and generate a new API key.

**Q: Can I use one license on multiple machines?**

Yes, the same API key works on all your devices.
