# Shadow Clone Team Templates

## Overview
This module contains predefined team templates for different project types. Each template defines master craftsmen teams with specific expertise areas, ensuring no weak links in the system.

## Software Development Masters

### Team Composition
- **Requirements Master**: User stories, specifications, stakeholder communication
- **Architecture Master**: System design, technical decisions, pattern selection
- **Implementation Master**: Core development, code craftsmanship
- **Testing Master**: Quality assurance, test automation, validation
- **DevOps Master**: CI/CD, deployment, infrastructure

### Specialization Notes
Each master is fully capable of handling the entire project independently but excels in their domain. When working together, they create synergy through coordinated expertise.

## Custom Team Configuration

### Dynamic Team Creation
Teams can be dynamically configured based on project needs:
```
team_composition = {
    "Custom Domain Team": agent_count,
    "Specialized Team": agent_count,
    "Integration Team": agent_count
}
```

### Agent Distribution Within Teams
- **2 agents**: Lead + Specialist
- **3 agents**: Lead + Implementation + QA
- **4 agents**: Lead + Senior Dev + Junior Dev + QA
- **5+ agents**: Additional specialists as needed

## Team Template Application

### Selection Criteria
1. **Project Type**: Match team template to project domain
2. **Complexity**: Scale team size based on project scope
3. **Timeline**: Adjust team composition for deadline requirements
4. **Special Needs**: Add specialists for unique requirements

### Template Customization
All templates can be customized while maintaining the core principle: every agent is a master craftsman, no weak links allowed.

## Integration with Agent Rules

### Rule Injection Protocol
Every agent in these teams receives:
1. **Core Agent Rules**: Universal behavioral DNA
2. **Role-Specific Rules**: Specialized enhancements
3. **Team Context**: Team-specific coordination protocols
4. **Project Rules**: Project-type specific guidelines

This ensures all agents operate with consistent excellence regardless of their specific role or team assignment.
