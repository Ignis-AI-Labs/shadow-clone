# Icon Requirements for VS Code Marketplace

## Icon Specifications
- **Size**: 128x128 pixels (PNG format)
- **Background**: Transparent or dark (#1e1e1e to match gallery banner)
- **Style**: Flat, modern design that scales well
- **File**: Save as `icon.png` in this directory

## Design Suggestions
1. **Shadow Clone Logo** - Use the main Shadow Clone branding
2. **AI Agent Symbol** - Abstract representation of multiple agents
3. **Ignis Labs Logo** - If using publisher branding
4. **Combination** - Shadow Clone + AI elements

## Color Palette
- Primary: #007ACC (VS Code blue) or brand colors
- Secondary: #FFFFFF (white) for contrast
- Background: Transparent or #1e1e1e

## Tools to Create Icon
- Adobe Illustrator/Photoshop
- Figma (free)
- Canva (free tier available)
- GIMP (open source)

## Example Icon Elements
- Multiple connected nodes (representing agents)
- Lightning bolt or energy symbol
- Code brackets with AI sparkle
- Shadow/clone visual metaphor

## Testing
Icon should be clearly visible and recognizable when displayed at:
- 128x128 (marketplace)
- 48x48 (extension list)
- 32x32 (activity bar)

Place final 128x128 PNG as `icon.png` in this directory.
